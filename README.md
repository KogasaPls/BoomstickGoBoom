# Boomstick Go Boom

A World of Warcraft addon that plays the cat that go "boom" sound when you hit boomstick.

## Commands

- `/bgb channel Master|SFX|Ambience|Music|Dialog` -- change the audio channel (default: Master)
- `/bgb test` -- play the sounds on the configured channel
